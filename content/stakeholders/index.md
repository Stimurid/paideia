# Каталог стейкхолдеров

В обстреле проекта от лица стейкхолдеров используются голоса из этого
каталога. Каждый голос — отдельный md-файл в этой папке.

## Группы

- **Внутренние:** студенты, преподаватели-консерваторы, преподаватели-инноваторы, методист, ректорат
- **Регуляторные:** методическое управление, ФГОС-эксперт, надзор по AI/данным, юрист
- **Внешние:** работодатели, родители, выпускники, академическое сообщество, рынок труда
- **Технические:** IT-служба, разработчики, vendor LLM
