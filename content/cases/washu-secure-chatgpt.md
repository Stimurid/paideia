## Описание

Secure WashU ChatGPT — изолированный GPT-4o инстанс, пригодный для HIPAA
и FERPA сценариев. Не подтягивает данные из веба, не делится с
внешними сторонами. Один из самых ранних кампусных сервисов с явным
compliance-фокусом (декабрь 2023).

## Что переиспользуемо

Для университетов с медицинской школой / исследовательскими данными
изолированный инстанс — обязательный первый шаг до открытого доступа к
обычным сервисам.
