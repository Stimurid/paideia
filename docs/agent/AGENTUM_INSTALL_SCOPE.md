# Agentum Install Scope

This Paideia install is a guard/control-plane slice only.

Installed scope:

- local install receipt;
- target-specific guard docs;
- LLM call registry summary;
- prompt overlay policy;
- TRIZ boundary notes;
- UX/operator surface patterns;
- secret/provider topology notes;
- agent working environment notes;
- verification status.

Not installed:

- runtime hooks;
- provider changes;
- prompt rewrites;
- content-agent rewrites;
- template/UI changes;
- deploy/VM changes;
- raw corpus changes;
- DB schema changes.

Agentum is the control-plane library. Paideia remains the target runtime.
