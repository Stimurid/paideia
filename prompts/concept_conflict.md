# Concept Conflict Detector

Тебе дана **пара концептов** из библиотеки Paideia. Оба близки по embedding —
это значит они говорят про похожую вещь. Твоя задача — **классифицировать
отношение** между ними.

## Входные данные

```
A: {name_a} (из «{book_a}» / автор {author_a})
   определение: {definition_a}

B: {name_b} (из «{book_b}» / автор {author_b})
   определение: {definition_b}
```

## Возможные отношения

- **`equivalent`** — это одно и то же понятие в разных формулировках.
  Можно склеить, никакого выбора не требуется.

- **`complements`** — концепты разные но **поддерживают друг друга**, говорят
  про разные грани одного. Можно использовать вместе.

- **`conflicts`** — концепты **противоречат**: предлагают несовместимые
  объяснительные рамки, школы, методологии, оценки. Использовать вместе
  без явного указания «вот два разных взгляда» нельзя.

- **`unrelated`** — близость embedding-а обманула, это про разные вещи.

## Если conflicts — назови школы

Когда `relation: conflicts`, обязательно укажи **`school_a`** и **`school_b`** —
короткие ярлыки позиций, к которым принадлежат концепты. Примеры:

- A=«стимул-реакция» / B=«зона ближайшего развития»
  → school_a: «бихевиоризм», school_b: «культурно-историческая теория»
- A=«готовая программа» / B=«эмерджентная программа»
  → school_a: «технократический подход», school_b: «коэмерджентный подход»
- A=«AI как инструмент учителя» / B=«AI как автономный агент-тьютор»
  → school_a: «инструментализм», school_b: «агентная автономия»

## Формат ответа

```json
{
  "relation": "equivalent|complements|conflicts|unrelated",
  "rationale": "1-2 предложения — что общего, в чём разница",
  "school_a": "ярлык позиции А (только если conflicts)",
  "school_b": "ярлык позиции Б (только если conflicts)"
}
```

## Правила

- Будь **строг к conflicts**: если можно склеить — это complements, не
  conflicts. Конфликт означает реально несовместимые рамки.
- Не путай **уровень детализации** с конфликтом: «общее понятие» vs
  «частный случай» — это complements.
- Школы — **на русском, короткие** (1-3 слова). Без университетских
  отделений, без имён авторов в ярлыке.

Выведи только JSON.
