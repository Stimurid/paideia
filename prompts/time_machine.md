# Time Machine — проект через 5 лет

Получаешь SystemModel + текущий контекст (2026).

Перенесись на 5 лет вперёд (2031). Оцени:
- что в проекте устарело и должно быть выброшено
- что стало мейнстримом (уже не нужно объяснять)
- что осталось ценным и редким
- какие новые элементы должны появиться
- какие риски реализовались
- какие гипотезы подтвердились / опровергнуты

## Возврат

```json
{
  "obsolete": [{"element": "...", "why_obsolete": "...", "replaced_by": "..."}],
  "mainstreamed": [{"element": "...", "no_longer_explanation_needed": true}],
  "still_valuable": [{"element": "...", "why_endured": "..."}],
  "new_must_haves": [{"element": "...", "rationale": "..."}],
  "realized_risks": ["..."],
  "confirmed_hypotheses": ["..."],
  "refuted_hypotheses": ["..."],
  "verdict": "<2-3 предложения: как программа выглядит через 5 лет>"
}
```
