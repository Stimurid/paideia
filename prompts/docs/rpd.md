# Сборщик Рабочей программы дисциплины (РПД)

Из канваса проекта собери черновик РПД по обобщённому российскому образцу.
Адаптируй язык под академический стиль методического управления.

## Структура РПД (заполни каждый блок)

```json
{
  "title": "<полное название дисциплины>",
  "level": "бакалавриат|специалитет|магистратура|аспирантура",
  "direction": "<код и название направления подготовки>",
  "credits": <число зачётных единиц>,
  "hours_total": <общее число академических часов>,
  "hours_lectures": <часов лекций>,
  "hours_seminars": <часов семинаров>,
  "hours_independent": <часов СРС>,
  "goals": ["<образовательная цель>", ...],
  "tasks": ["<задача>", ...],
  "competencies": [{"code": "УК-1|ОПК-1|ПК-1|...", "name": "...", "indicators": ["..."]}],
  "learning_outcomes": [{"type": "знать|уметь|владеть", "text": "..."}],
  "thematic_plan": [
    {"week": 1, "topic": "...", "lecture_h": N, "seminar_h": N, "independent_h": N, "ai_role": "<если используется AI на этой неделе>"}
  ],
  "assessment": {
    "current": ["<формы текущего контроля>"],
    "intermediate": ["<рубежного>"],
    "final": "экзамен|зачёт|защита проекта"
  },
  "ai_governance": {
    "permitted_tools": ["..."],
    "prohibited_uses": ["..."],
    "transparency_requirements": ["<что студент должен фиксировать>"],
    "academic_integrity_addendum": "..."
  },
  "literature_main": ["<источник>"],
  "literature_additional": ["..."],
  "technical_support": ["<требования к платформе/инфраструктуре>"]
}
```

Запреты: не выдумывай нормативные коды — пиши «требуется уточнить» если не знаешь.
