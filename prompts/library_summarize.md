# Library Summarize — извлечь summary + topics книги

Получаешь:
- title, authors, year (если есть)
- первые ~15 000 символов текста (intro + начало книги)
- сэмплы из середины и конца

Верни **summary + topics + concept hints** для каталога библиотеки.

## Возврат: строгий JSON

```json
{
  "summary": "<2-4 абзаца на русском: о чём книга, основная мысль автора, для кого, методология>",
  "topics": ["образование", "методология", "Выготский", "ZPD", "..."],
  "key_concepts_preview": ["концепт 1", "концепт 2", ...],
  "intended_audience": "<кому полезна — методологи / преподаватели / исследователи / администрация>",
  "depth_level": "intro | mid | advanced | reference",
  "language": "ru | en | mixed",
  "estimated_value_for_paideia": "high | medium | low",
  "why_useful_for_paideia": "<1-2 предложения: как эта книга применима в проектировании образовательных программ с AI>"
}
```

Запреты:
- Не пиши «всеобъемлющий труд», «фундаментальная работа» — конкретные тезисы.
- topics — 5-12 точных, не общие слова.
- key_concepts_preview — 5-10 концептов которые **скорее всего** будут извлечены в extraction.
- summary НЕ должен быть рекламной аннотацией. Он должен помочь понять стоит ли читать.
