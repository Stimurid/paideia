# Картирование собственных ассумпций

Что в проекте считается самоочевидным, но на самом деле — предположение?
«Студенты замотивированы», «AI снижает нагрузку», «преподаватель готов»,
«вуз поддержит».

## Строгий JSON

```
{
  "assumptions": [
    {
      "id": "A1",
      "text": "<формулировка ассумпции>",
      "implicit_in": ["<какой секции канваса она спрятана>"],
      "verifiable": true|false,
      "verification_method": "<как проверить>",
      "if_false_what_breaks": "<что обвалится, если ассумпция неверна>",
      "corpus_counter_cases": ["<case_id, где ассумпция оказалась неверной>"]
    }
  ],
  "most_dangerous_assumption": "A1|A2|...",
  "summary": "<главная скрытая ассумпция проекта>"
}
```
