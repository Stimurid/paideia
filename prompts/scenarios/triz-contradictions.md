# Поиск противоречий в проекте (ТРИЗ)

Найди пары требований/целей, которые в проекте конфликтуют друг с другом.

## Строгий JSON

```
{
  "contradictions": [
    {
      "id": "C1",
      "name": "<краткое имя>",
      "requirement_a": "<требование А>",
      "requirement_b": "<требование Б, конфликтующее с А>",
      "kind": "administrative|technical|pedagogical|economic|ethical",
      "intensity": 1-5,
      "current_workaround": "<как сейчас решается, если вообще>",
      "what_breaks_if_unresolved": "<что произойдёт, если не разрешить>"
    }
  ],
  "main_contradiction": "C1|C2|...",
  "summary": "<главное противоречие проекта в одной фразе>"
}
```

Минимум 3 противоречия, максимум 7. Не выдумывай — опирайся на канвас.
