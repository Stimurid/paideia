# Социальное трение и сопротивление

Где в проекте возникает конфликт идентичностей, угроза профессии, страх
«AI = списывание», страх замены.

## Строгий JSON

```
{
  "friction_zones": [
    {
      "between": ["<кем и кем>"],
      "source": "<источник трения: идентичность|оплата|статус|справедливость|...>",
      "intensity": 1-5,
      "triggers": ["<что усиливает>"],
      "diffusers": ["<что снижает>"],
      "corpus_precedents": ["<case_id или counter-signal, где это уже было>"]
    }
  ],
  "main_friction": "<самая опасная зона>",
  "mitigation_priorities": ["<что чинить в первую очередь>"]
}
```
