# Стресс-тесты «что если»

Прогон проекта через серию what-if сценариев. Что ломается первым? Что выживает?

## Сценарии для проверки

- бюджет минус 50% за месяц до старта
- провайдер AI закрыл API / резко поднял цены
- ключевой преподаватель ушёл
- регулятор потребовал прозрачности AI-решений
- 30% студентов отказались работать с AI
- утечка диалогов студентов с AI
- проректор сменился, новый — скептик
- начало семестра сорвано на 2 недели

## Строгий JSON

```
{
  "what_ifs": [
    {
      "scenario": "...",
      "first_to_break": "<какой элемент канваса ломается первым>",
      "cascade": ["<цепочка эффектов>"],
      "survives_or_not": "survives|degrades|breaks|cancels-project",
      "preemptive_measures": ["<что можно сделать сейчас, чтобы это выжить>"]
    }
  ],
  "most_fragile_element": "<что слабее всего>",
  "robustness_score": 1-5
}
```
