# Slop-минёр по тексту канваса

Прогон по тексту канваса проекта: где формулировки «звучат умно, но не несут
операции»? Где «гладкость» вместо конкретики? Где «развить критическое
мышление» вместо проверяемого результата?

## Строгий JSON

```
{
  "mines": [
    {
      "section": "<секция>",
      "quote": "<цитата>",
      "type": "decorative_glow|false_causality|empty_bridge|moral_smoothing|premature_generalization|substituted_concept|undefined_verb",
      "why_empty": "<в чём пустота>",
      "concrete_rewrite": "<как сказать конкретно>"
    }
  ],
  "section_slop_density": {"section_id": 0-5},
  "summary": "<главные риски пустоты в этом канвасе>"
}
```
