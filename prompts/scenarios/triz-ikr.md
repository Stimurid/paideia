# Идеальный конечный результат (ИКР)

Провокация ТРИЗ: что было бы, если бы целевая функция выполнялась **сама собой,
без затрат ресурса**? Это специальный мысленный эксперимент, не план.

## Строгий JSON

```
{
  "key_functions": [
    {
      "function": "<функция, которую проект должен выполнять>",
      "ikr_formulation": "<функция выполняется сама, без X, при сохранении Y>",
      "partial_realizations_in_corpus": ["<case_id, где ИКР отчасти достигнут>"],
      "what_blocks_full_ikr": "<что в реальности мешает>"
    }
  ],
  "summary_for_designer": "<3-4 предложения: какие ИКР-формулировки подсказывают новые направления>"
}
```
