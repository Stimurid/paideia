# Мутация под локальный контекст

Адаптация архитектуры проекта под конкретный контекст. Если в канвасе контекст
не задан явно — спроси через open_questions. Если задан — дай конкретный план
мутации.

## Строгий JSON

```
{
  "context_inferred": {
    "region": "<откуда видишь>",
    "scale": "<студентов, ресурсов, бюджета>",
    "constraints": ["<нормативные|экономические|культурные>"]
  },
  "mutations": [
    {
      "what_changes": "<что меняем>",
      "from": "<как было>",
      "to": "<как становится>",
      "why_needed_for_context": "<почему именно в этом контексте>"
    }
  ],
  "preserved": ["<что сохраняется>"],
  "open_questions": ["<что нужно уточнить у пользователя>"]
}
```
