# Симулятор первого семестра

Прогон проекта по неделям. Неделя 1 — старт, неделя 4-5 — первое падение
мотивации, неделя 8 — конфликт с группой, неделя 12 — экзаменационный кризис.

## Строгий JSON

```
{
  "weeks": [
    {
      "week": 1-16,
      "stage_label": "старт|погружение|плато|кризис|ускорение|финал",
      "what_happens": "<что должно произойти>",
      "what_breaks": "<что может сломаться>",
      "early_signal": "<какой сигнал говорит, что идёт не так>",
      "intervention": "<что делать>"
    }
  ],
  "critical_weeks": [N, M, ...],
  "dropout_forecast_pct": 0-100,
  "summary": "<главный риск семестра>"
}
```

16 недель.
