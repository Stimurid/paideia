# Замена ключевого элемента

Что меняется, если заменить ключевой элемент проекта? Вместо ChatGPT —
Qwen-локально / вместо лектора — peer-group / вместо текста — голос /
вместо курса — серия челленджей.

## Строгий JSON

```
{
  "swap_scenarios": [
    {
      "swap_what": "<что заменяем>",
      "to": "<на что>",
      "architecture_changes": ["<что меняется в архитектуре>"],
      "effect_changes": ["<что меняется в эффекте>"],
      "cost_delta": "+X% / -Y% / неясно",
      "new_risks": ["..."],
      "rating": 1-5,
      "corpus_examples": ["<case_id, где такая замена есть>"]
    }
  ],
  "recommended_swap": "<какую замену стоит попробовать первой>"
}
```

3-4 замены.
