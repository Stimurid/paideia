# Что было бы дальше: 3 вектора эволюции

Применить эволюционные паттерны корпуса и предсказать 3 следующих шага для
проекта. С опорой на кейсы, которые «прошли этот путь раньше».

## Строгий JSON

```
{
  "vectors": [
    {
      "name": "<имя вектора>",
      "direction": "<куда движение>",
      "intermediate_steps": ["<этапы>"],
      "endpoint_in_5_years": "<что станет>",
      "corpus_precedents": ["<case_id, кто прошёл этот путь>"],
      "main_risk": "<что обычно ломается>",
      "likelihood": 1-5
    }
  ],
  "recommended_vector": "<какой из 3 выбрать и почему>"
}
```
