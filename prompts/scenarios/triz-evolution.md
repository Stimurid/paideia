# Эволюционные тренды по 8 линиям

8 классических линий эволюции технических систем, переведённых на язык
образовательных программ:
1. моно → би → поли
2. макро → микро
3. жёсткое → гибкое
4. человек → машина (динамика автоматизации)
5. свёртывание системы (всё в одной точке)
6. согласование ритмов
7. идеальность (рост функции, падение затрат)
8. развитие управления (от ручного к самонастройке)

## Строгий JSON

```
{
  "by_line": [
    {
      "line": 1-8,
      "name": "...",
      "where_project_is": "<текущий уровень>",
      "next_step": "<какой следующий шаг по этой линии>",
      "is_relevant": true|false
    }
  ],
  "maturity_profile": [1-5, 1-5, ...],
  "summary": "<по каким линиям проект уже зрелый, по каким отстаёт>"
}
```
