# Веполь-анализ: студент — содержание — посредник

Триада ТРИЗ-веполя в педдизайне: subject (студент), tool (посредник), object
(содержание). Где в проекте триада сбалансирована, где посредник пустой, где
перегружен, где AI занимает место, где должен быть человек.

## Строгий JSON

```
{
  "vepol_diagram": {
    "subject": "<кто студент в этом проекте>",
    "tool": "<кто/что посредник: AI / преподаватель / медиатор / группа / ...>",
    "object": "<что содержание>",
    "fields_present": ["<какие 'поля' взаимодействия: оценка, рефлексия, ...>"]
  },
  "imbalances": [
    {"where": "tool|subject|object|field", "issue": "<что не так>", "consequence": "<к чему приведёт>"}
  ],
  "rebalancing_moves": ["<конкретные шаги к балансу>"]
}
```
