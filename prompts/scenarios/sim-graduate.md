# Симулятор выпуска

Что реально приобретут студенты к концу программы? Заявленные vs реалистично
формируемые компетенции.

## Строгий JSON

```
{
  "declared_competencies": [
    {
      "name": "<заявленная компетенция>",
      "realistically_formed": "fully|partially|knowledge_only|illusory",
      "what_actually_forms": "<что на самом деле>",
      "what_blocks": "<что мешает полному формированию>",
      "fix_if_any": "<что подкрутить>"
    }
  ],
  "unexpected_competencies": ["<что сформируется как побочный эффект>"],
  "gap_with_market": "<разрыв с тем, что нужно работодателям/обществу>",
  "summary": "<честная одна фраза о выпускнике>"
}
```
