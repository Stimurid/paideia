# Обстрел проекта от лица стейкхолдеров

Для каждой ключевой позиции — 5 самых неприятных вопросов к проекту.

## Строгий JSON

```
{
  "attacks": [
    {
      "stakeholder": "...",
      "voice": "<как этот стейкхолдер говорит, его тон>",
      "questions": [
        {"q": "<вопрос>", "intent": "<что он хочет услышать>", "current_answer_ok": true|false, "if_not_ok_why": "..."}
      ]
    }
  ],
  "weakest_position_of_project": "<какие вопросы проект не выдерживает>",
  "must_prepare_answers_for": ["<топ-вопросов, без которых нельзя выходить>"]
}
```

5 вопросов на стейкхолдера, минимум 4 стейкхолдера.
