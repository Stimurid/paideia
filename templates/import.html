Импорт кейсов

Один URL → один кейс. Файл/большой текст (PWC-отчёт, ТюмГУ-презентация, PDF) → много кейсов одним прогоном.

Загрузи отчёт (PDF · DOCX · PPTX · MD · TXT) или вставь длинный текст. Документ режется на сегменты по эвристическим заголовкам или фиксированным чанкам ~3500 символов; каждый сегмент прогоняется через gpt-4.1-mini на извлечение кейсов. Лимит: 30 сегментов за прогон (большой PDF делится по частям).
EPUB здесь пока не поддерживается — для него /library/upload.

Найдено кандидатов:

    Один URL новости/официальной страницы/публикации. Sonar-pro проходится по источнику, извлекает кейс по канвасу, возвращает draft md. Ты ревьюишь и сохраняешь.

    не извлечено / нужно уточнить
      другие кандидаты в источнике
