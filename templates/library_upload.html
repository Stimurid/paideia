📥 Загрузить книгу

PDF · DOCX · PPTX · EPUB · MD · TXT. До 50 МБ. Без лимита по символам — режется на чанки по 700 символов с overlap.
Книга становится частью библиотеки навсегда: summary + topics + 3-8 концептов на пару секций + векторный индекс. Для разового разговора о документе лучше главная «У меня проект». Для россыпи кейсов из одного файла — /import.

После upload запустится pipeline:
  1. Извлечение текста (1-5 с)
  2. Разбиение на секции + чанки (1 с)
  3. LLM-summary + topics (~15 с)
  4. LLM-extraction концептов (по 2 секции → ~5 с/пара)
На книге 200 страниц весь pipeline занимает 3-7 минут.
